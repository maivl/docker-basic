This project is intended to get my feet wet with Docker.
The project contains a Dockerfile which is built into the Image named "ydubler/nodetest:1" using the command:
docker build -f Dockerfile -t ydubler/nodetest:1 .
Then the ydubler/nodetest:1 Image is used to create a running container, which is the Node server serving up this website, using the command:
docker run -p 3000:3000 ydubler/nodetest:1
If you see this website, then the project is a success.
